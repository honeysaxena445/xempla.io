Xempla Assessment - ESG Dashboard Prototype
This project is a prototype of an ESG (Environmental, Social, and Governance) Compliance Scorecard for building management, created as part of the assessment for the Data Analyst role at Xempla.

Objective: To create a simple dashboard that tracks and visualizes the ESG performance of different buildings over time.

Tools Used:

Data Preparation: Microsoft Excel / Google Sheets

Dashboarding: Microsoft Power BI
